# Claude Code Configuration

This project is configured with Claude Code support for educational/documentation work.

## Enabled Plugins

- **system** — Audit, cleanup, setup, status
- **workflow** — Explore, plan, next, ship
- **memory** — Handoff, memory management
- **development** — Analyze, review, test, fix, run, git

## Available MCP Servers

- **Serena** — Semantic code understanding (70-90% token reduction)
- **Context7** — Up-to-date library and framework documentation

## Quick Start

Run `/help` to see available commands and skills.

- `Spec-Driven/` — Main tutorial/learning content
- `.claude/` — Claude Code configuration
- `.omc/` — OMC orchestration state
